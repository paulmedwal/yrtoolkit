Mood Counter

1. Introduction

Last year, NPR’s Morning Edition ran a story about an app called Mood Ring, created by our friends at Youth Radio, using App Inventor. Mood Ring invites teens to track how they’re feeling using any combo of nine emojis, and to reach out to trusted others when they’re having a rough time. Youth Radio’s Storm White, who co-designed Mood Ring with some other Youth Radio members, told NPR her team wanted to develop “a playful way to address something serious.”

Let's make an app that can keep track of how many times you feel different emotions throughout the day.

2. Add components

First, start by adding some components. You need a Label (with its text set to "Snapchat Remix", Button, Canvas (with its height and width set to "Fill parent"), and Camera.

Make other changes to the component properties that you want.

3. Add click handler

Click on the Blocks button on the right side top bar.

Click on the happyButton drawer on the left and drag the When happyButton.Click block workspace.

4. Add set angryLabel.Text

Click on the happyLabel drawer, then drag the set happyLabel.Text to block to the when happyButton.Click block.

5. Add __ + __

Click on the Math drawer. Click and drag the __ + __ block and add it to the set happyLabel.Text to block.

6. Add happyLabel.Text

Click on the angryLabel drawer. Click and drag the angryLabel.Text block and add it to the first socket of the __ + __ block.

7. Add 1

Click on the Math drawer. Click and drag the 0block and add it to the second socket in the __ + __ block. Change the "0" to a "1".

8. Duplicate blocks

Try clicking the different emojis. What happens now?

Let's now make the angry emoji work too. Right click (or ctrl/control click) on the when happyButton.Click block, then click Duplicate.

9. Change "happyButton" to "angryButton"

Click on the "happyButton" drop down on the when happyButton.Click block and choose "angryButton"

10. Repeat for happyLabel

Change the dropdown of set happyLabel.Text to to set angryLabel.Text to. And change happyLabel.Text to angryLabel.Text

11. Repeat for sadButton

Repeat steps 7, 8, and 9 for the sadButton.

12. Test it out!

Try it out on your device. Tap on the different emojis. Do the numbers increment when you tap on the buttons?

Change up this basic tutorial by adding more features using the ideas in the next section.

13. Expand your app

Show an inspiring quote based on how the user feels.

Set the label component's text to a random message after the user chooses their mood.

Play a song or display an image to cheer up a friend.

Use the player component to play a song after the userxw selects an emotion or the image component to show a picture.

Personalize the app with the user’s name.

Use the text input component to get the user’s name and display it in the app.

Text a friend when the user is feeling sad.

Use the texting component to send a message when the sad button is clicked.

Let users review how they felt in the past.

You can use the file component to store what happens each time the user hits the button.
