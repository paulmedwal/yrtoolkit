Healthy Plant tutorial

1. Setup

We'll make an app that measures the health of a plant using graphs.

Click here to learn how to set up your device if you haven't yet.

2. Add three VerticalArrangement components

Click on the Layout and drag out three VerticalArrangement to HorizontalArrangement2.

3. Rename your VerticalArrangements

Select VerticalArrangement1, click "Rename" and rename it to "LightBar". Then, rename VerticalArrangement2 to "MoistureBar" and VerticalArrangement3 to "TemperatureBar".

4. Set height to "0px" and width to "Fill Parent"

Set the height of LightBar to "0px" and set the width to "Fill Parent". Repeat for MoistureBar and TemperatureBar.

5. Set the BackgroundColor

Select LightBar and set the BackgroundColor to yellow. Set the BackgroundColor of MoistureBar to blue and the BackgroundColor of TeperatureBar to red.

6. Add set LightBar.Height to

Click on Blocks.

Click LightBar1 and select set LightBar.Height to. Connect it to the first socket of the __ x 0.75 block.

7. Connect set LightBar.Height to to when Arduino101LightSensor1.LightSensorDataReceived

Locate the when Arduino101LightSensor1.LightSensorDataReceived (it'll have a few blocks inside of it). It should look like this:


Connect your whole set LightBar.Height to to when Arduino101LightSensor1.LightSensorDataReceived (under the blocks already inside).

8. Add get intensity

On the when Arduino101LightSensor1.LightSensorDataReceived block, hover over the red "intensity". Click on get intensity in the pop-up and connect it to the first socket of the __ x 0.75 block.

9. Repeat steps 6-8 for MoistureBar

Get the set MoistureBar.Height to block, and it to __ x 4. Then, add blocks to when Arduino101Moisture1.MoistureReceived. Hover over "moisture" and connect get moisture.

10. Repeat steps 6-8 for TemperatureBar

Get the set TemperatureBar.Height to block, and it to __ x 0.75. Then, add blocks to when Arduino101Humidity1.TemperatureReceived. Hover over "temperature" and connect get temperature.

11. Test it out!

Try it out on your device. What happens to the bars? Your app should look something like this:
