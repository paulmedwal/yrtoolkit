The Challenge

Youth Radio ran a story about a game called Escape Room. Escape room invites people to get off their phones and play a real-video game where they have to solve puzzles in order to espace a room. Youth Radio's article explores the creation of Escape Rooms and interviews how they're run. One business owner says, "Everyone has to put their phones away and everyone has to interact with each other."

Now it’s your turn to work with App Inventor to design and build your own version of an Escape Room app that uses a cipher and translation tool. How else can you get people thinking about puzzles through your app? We can’t wait to see what you come up with!

Click the tutorial panel to start building your app!

Let's add some features to the translation screen.

If you're not already there, go to Screen2.

Click on translate in the Blocks panel.

Click on YandexTranslate1.

Drag out the "call YandexTranslate1.RequestTranslation" block and connect it.

Expand your app

Shake to move to the next level of your game.

Use the accelerometer sensor sense when the device is being shaken in order to move to the next level.

Compare two words.

Use a comparison block to check to see if a word is correct. You can use this to create your own code and see if the user got the correct answer.

Morse Code!

Using the comparison technique above, create your own Morse code and check to see if a word is correct.

Use the encrypt procedure.

In Screen1, there's an encrypt procedure that can be used to encrypt your own text.

Create a drop-down menu of options.

Use the spinner component to give the uver multiple options to chose from. You can also attach different actions to each option.

Use the Video Player.

Enhance your app by using the video player component to use more media and ask more complicated or interesting questions.

Give the user disappearing alerts.

The Notifier component gives the user alerts at a particular event, which can be used to give hints or show messages to the user.

Create options with CheckBoxes.

Enhance your app by using the checkbox component to use more media and ask more complicated or interesting questions.

Make your text speak!

Make your app a bit more vocal with the TextToSpeech component. Try using this with the Yandex Translator.
