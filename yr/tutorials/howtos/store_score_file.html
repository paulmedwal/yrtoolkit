Add a File Component

From the Storage Palette in the Designer, drag in a File component.

Append to File vs Save to File

In the Blocks Editor, open the File drawer. There are two similar blocks: File.AppendToFile and File.SaveFile.

Saving to a file means overwriting a file if it already exists with the new text.

Appending to a file means adding new text to the file if it already exists.

Appending Score to File

Drag out the File.AppendToFile block and two empty text blocks.

Fill in the fileName field with one of the empty text blocks, and enter scores.txt.

Drag in a join block from the Text drawer, and snap it into the text field.

Fill the second empty text box with Score: , and snap it into the join block.

Finally, snap in a get global score block.
