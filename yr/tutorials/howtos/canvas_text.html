1. Add a Button

In Designer, go to User Interface and add a Button to your Viewer. Change the text of the button in the Properties panel if you want.

2. Add a Textbox

Again in User Interface, add a TextBox to your Viewer.

3. Add when Button1.Click

Click Blocks.

Click on Button1 and drag out when Button1.Click.

4. Add call Canvas1.DrawText

Click on Canvas1 next and connect call Canvas1.DrawText.

5. Add TextBox1.Text

Click TextBox1 and add TextBox1.Text to the "text" socket of the call Canvas1.DrawText block.

6. Add two 0 blocks

Next, click on Math and drag out two 0 blocks and connect them to the "x" and "y" sockets.

7. Change block values

Change the values of both the 0 blocks to "10" (or anywhere else on the screen you want the blocks to be).

8. Test out your app!

Once it works, you can move around the text or let the user control where the text goes.
