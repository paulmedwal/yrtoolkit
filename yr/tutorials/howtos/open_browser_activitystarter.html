Add an ActivityStarter Component

In the Designer window, open the Connectivity Palette, and drag in an ActivityStarter component.

In the Properties pane, set the Action to android.intent.action.VIEW.

Add Link to Activity Starter

Switch to the Blocks Editor. From the ActivityStarter drawer, drag out a set ActivityStarter.DataUri block.

Drag in a new empty text block, and snap it into the set block. Fill this text block in with the URL you want to open.

Note: You can also do this in the Properties pane of the Designer window.

Calling the ActivityStarter

From the ActivityStarter drawer, drag in a call ActivityStarter.StartActivity block.

Snap it in under where you set the DataUri. That's it!
