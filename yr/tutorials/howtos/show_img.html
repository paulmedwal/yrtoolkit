1. Add an Image

In User Interface, click and drag an Image into the Viewer.

2. Add Image source

Go to the Properties panel and select "dog.jpg" or upload an image of your own. You should see the image show up on your device screen. Now unclick "Visible". You'll add in Blocks to make your image show up when the button is pressed.

3. Add a Button

Still in User Interface, click and drag a Button into the Viewer.

4. Add when Button1.Click

Click Blocks.

In Blocks, click on Button1 and drag out the when Button1.Click block.

5. Add set Image1.Visable to

Click Image1 and connect the set Image1.Visable to block to when Button1.Click.

6. Add true

Click Logic and connect a true block to the set Image1.Visable to block.

Test out your app! On the click of the button, does the image show up?
