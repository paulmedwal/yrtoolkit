Add Components

In the Designer, from the User Interface Palette, drag in a Button component.

From the Media Palette, drag in a TextToSpeech component.

Add Button Click Block

In the Blocks Editor, from the Button drawer, drag in a when Button.Click block.

Add Speak Block

From the TextToSpeech drawer, drag in a TextToSpeech.Speak block.

Add Text Block

From the Text drawer, drag in an empty text block. Fill in this text block with any text of your choice.

Test!

Connect your device, and click on the button. Your app should 'speak' whatever text you entered!
