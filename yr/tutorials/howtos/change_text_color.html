Add a Label Component and Change Color via Properties

From the User Interface Palette of the Designer, drag in a Label.

In the Properties pane, there is a TextColor field. Clicking on this will reveal a palette of colors to choose from.

Change Color via Blocks

We can change text color via the Blocks Editor as well.

From the Label1 drawer, drag in a set Label.TextColor block.

From the Colors drawer, drag in any color and snap it in.

Making New Colors

The Make Color block allows any combination of three numbers, where the numbers are in the range of 0-255. What does this mean?

Every color that we see is a combination of red, green, and blue, so every color can be described via an RGB value. The Make Color block does exactly that--take three R, G, and B values to create a new color.

0 means none of a certain color, and 255 is the max of a certain color.

This is a color wheel. It shows how different amounts of R, G, and B, respectively, create new colors.

Here are some common colors and their RGB values to get you started on making your own custom colors.

  • Red: 255,0,0
  • Green: 0,255,0
  • Blue: 0,0,255
  • White: 255,255,255
  • Black: 0,0,0
  • Yellow: 255,255,0
  • Magenta: 255,0,255
  • Cyan: 0,255,255
