Map Component

Click and drag a Map to your Viewer.

There are a lot of options here! Let's go through what some of the properties mean.

Map-Specific Properties

CenterFromString expects a latitude/longitude pair that the map centers at.

EnablePan allows a user to drag the map around, or makes it stay static.

Similarly, EnableRotation lets the user spin the map around.

EnableZoom allows the user to zoom in and out by pinching the map.

LocationSensor allows you to connect to a separate LocationSensor component.

MapType allows you to select three map appearances: Roads, Aerial, and Terrain.

ShowCompass shows a compass directly on the map.

ShowUser shows the user's auto-detected location on the map.

ShowZoom shows +/- zoom buttons directly on the map.

ZoomLevel controls how zoomed in or out the map is.

Event Blocks

Here's a quick overview of the available event blocks.

"Feature" blocks (like when Map1.FeatureClick) allow you to interact with any feature on the Map--usually a Marker.

Tap/Click blocks (like double tap, long press at point, and a few others) gauge various types of click events that can happen on the map.

Procedure Blocks

Here's a quick overview of the available procedure (call) blocks.

CreateMarker will let you create a new Marker, and returns the new Marker. Potentially, you could save this new Marker in a list to use later. Can you think of a reason why you might do that?

FeatureFromDescription and LoadFromURL are advanced and you probably won't find yourself using them much.

PanTo allows you to go to a different lat/long and zoom, anywhere on the map. It's a good way to quickly change where the map is looking at. Hint: It's very similar to changing the CenterFromString property.

Save lets you save the current map to your device storage.

The available getters and setters are the same as in any other component, so they are omitted here.
