Add a Texting Component

In the Designer, from the Social Palette, drag in a Texting component.

Set Phone Number & Message Text

Switch to the Blocks Editor. From the Texting drawer, drag out a set Texting.Message block and a set Texting.PhoneNumber block.

Drag out two empty text blocks and snap them in, filling them out with the desired phone number to text and the desired message.

Send Message

Drag out a Texting.SendMessage block and snap it in!
