1. Add a Button

In Designer, add a Button to your Viewer.

2. Change button text

Change the text of your button so the user knows which color they're selecting. You can also change the size, color, and shape of the buttons.

3. Add when Button1.Click

Click Blocks.

Click Button1 and drag out the when Button1.Click block.

4. Add set Canvas1.PaintColor to

Click on Canvas1 and choose set Canvas1.PaintColor to and connect it.

5. Add a blue block

Click on Colors and choose a blue block and connect it to set Canvas1.PaintColor to.

6. Test out your app!

Once it works, add more buttons to give the user more options for colors.
