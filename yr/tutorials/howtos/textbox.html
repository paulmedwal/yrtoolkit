1. Delete russianText

First, delete the russianText label (you don't need it anymore!).

2. Add a TextBox

Click and drag a TextBox to your Viewer.

3. Add TextBox1.Text

Go to Blocks.

Click on TextBox1 and add TextBox1.Text to the textToTranslate socket of call YandexTranslate1.RequestTranslation.

4. Change the language code

In the text block that reads "ru-en" change the text inside to "it-en" (for Italian to English). I know the phrase, "Vuoi ballare con me?" in Italian and want to translate it using my app.

5. Test it out!

Change the language code (in Blocks) to another language. Does the translator correctly translate your text?
