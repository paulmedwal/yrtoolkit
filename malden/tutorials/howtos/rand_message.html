1. Add a Label and a Button

In Designer, go to User Interface and add a Label. Then, go to User Interface again and add a Button.

2. Add when Button1.Click

Click Blocks.

In Blocks, click on Button1 and drag out the when Button1.Click block.

3. Add set Label1.Text to

Next, click on Label1 and add the set Label1.Text to block when Button1.Clicks.

4. Add pick a random item list

Click on Lists and add the pick a random item list block to set Label1.Text to.

5. Add make a list

Click on Lists again and add the make a list block to pick a random item list.

6. Add another item to the list

Click on the blue gear of the make a list block and click and add another list item to the block.

7. Attach Text blocks to sockets

Click on Text and drag and attach a " " blocks to each socket of make a list. Change the text of each " " block to an encouraging statment, such as "You got this!", "I believe in you", or "You can do it!"
